Rectangle dodger
================
![alt text](http://imgur.com/z4BdZj0 "Game screenshot")

Description
-----------
Adventures of white rectangle.


Instructions
------------
Avoid non-white rectangles.

Mouse - Move
Keys:
Escape - Quit
R - Restart


Installation
------------

	$ git clone https://github.com/mat3k/rectangle_dodger.git
	$ gem install gosu


Run
---

	$ ruby main.rb
