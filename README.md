Rectangle dodger
================
![Game screenshot](http://imgur.com/z4BdZj0)

Description
-----------
Adventures of white rectangle.


Instructions
------------
Avoid non-white rectangles.


Controls
--------
* Mouse - Move
Keys:
* Esc - Quit
* R - Restart


Installation
------------

	$ git clone https://github.com/mat3k/rectangle_dodger.git
	$ gem install bundler
	$ bundle install


Run
---

	$ ruby main.rb
