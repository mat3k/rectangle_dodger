Rectangle dodger
================
[Screenshot](http://i.imgur.com/z4BdZj0.png)

Description
-----------
Adventures of white rectangle.


Instructions
------------
Avoid non-white rectangles.


Controls
--------
* Mouse - Move
* Esc - Quit
* R - Restart


Installation
------------

	$ git clone https://github.com/mat3k/rectangle_dodger.git
	$ gem install bundler
	$ bundle install


Run
---

	$ ruby main.rb
